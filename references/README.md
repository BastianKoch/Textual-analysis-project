# References & Documentation

This folder contains reference materials, data dictionaries, manuals, and other supporting documentation for the project.

## Contents

### Data Dictionaries
- Definitions of variables and columns in your datasets
- Data types and formats
- Missing value conventions
- Example: `earnings_calls_data_dictionary.md`

### Manuals & Guides
- Data source documentation
- How data was collected/processed
- Dataset limitations and caveats
- Column descriptions

### Academic Papers
All research papers and literature are stored in our shared **Dropbox folder**:

**[Textual Analysis Literature on Dropbox](insert-dropbox-link-here)**

Papers are organized by topic:
- `sentiment-analysis/` - Papers on sentiment analysis techniques
- `nlp-earnings-calls/` - NLP work on earnings calls and corporate text
- `text-classification/` - Document classification and topic modeling
- `feature-extraction/` - Text feature engineering methods

### Bibliography & Citations
- `References.md` - Full bibliography with links (in Dropbox)
- `bibliography.bib` - BibTeX format for citations in papers/reports

## How to Use This Folder

**Local repo (GitHub):**
- Keep metadata, data dictionaries, and notes here
- Keep file sizes small
- Link to papers on Dropbox

**Shared Dropbox:**
- Store actual PDF files
- Maintain organized folder structure
- Update bibliography as you add papers

## Contributing

When you read a paper:
1. Add an entry to `References.md` in the Dropbox folder
2. Include: Authors, year, title, link, and your notes
3. Organize in the appropriate topic subfolder

Example format:
```markdown
## Sentiment Analysis

1. **Sentiment Analysis in Financial Texts** (2023)
   - Authors: Smith, J. & Johnson, K.
   - DOI/Link: https://doi.org/10.xxxx/xxxxx
   - File location: `sentiment-analysis/Smith_2023.pdf`
   - Relevance: Methods for analyzing earnings call sentiment
   - Key findings: Use of domain-specific lexicons improves accuracy by 12%
```
